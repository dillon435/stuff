Trivia

  • Nathan Fillion, Adam Baldwin, and Alan Tudyk all appeared in Firefly together
  • During development, the developers wanted it to be raining during the night time New Mombasa street segments. Unfortunately, the engineers weren't able to add in rain bcause the game enigine couldn't handle it. But when they remastered the game for MCC, they added in the rain effect, making the game the way it was meant to be
  • The game was originally called "Halo 3 Recon," but Bungie changed the name to Halo 3 ODST because the players would be playing as ODSTs in game
  • Halo 3 ODST is the only Halo game in the franchise to solely take place on Earth
  • While the title says Halo 3, the game actually takes place during Halo 2
  • Later on into the Halo franchise, Buck becomes a Spartan, and returns in the main campaign oh Halo 5
